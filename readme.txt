== Description ==

Custom Portfolio Type with filtering without page refresh Using Shortcode [filter-portfolio]


Primary Features
*  Very easy to use just copy this shortcode [filter-portfolio]
*  A front-end Custom Post Filtering
*  Featured Images (thumbnails)
*  Developer-friendly (actions, filters, templates)
*  This plugin is free


== Installation ==

Go to Plugins > Add New.
Search for "Portfolio With Filter".
Click "Install Now".

OR

Download the zip file.
Upload the zip file via Plugins > Add New > Upload.
Activate the plugin. Look for "Portfolio" in the admin menu.

= What is the shortcode? =

use this shortcode:

[filter-portfolio]
